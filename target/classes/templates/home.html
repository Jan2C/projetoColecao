Projeto de Coleção de Livros

Sistema de Controle de Acesso

Bem-vindo ao Sistema de Controle de Acesso de uma coleção de livros. Este sistema foi desenvolvido utilizando Spring Boot e Java, proporcionando uma solução segura e eficiente para gerenciar livros e escritores.

Recursos Principais:

  • Autenticação de livros/escritores
  • Listagem de livros/escritores
  • Cadastro de livros/escritores
  • Alteração de livros/escritores
  • Exclusão de livros/escritores

Este sistema é uma tentativa de aplicar na prática a teoria de desenvolvimento de sistemas através de um hobbie que eu tenho.

Ano da tentativa 2022-2023
